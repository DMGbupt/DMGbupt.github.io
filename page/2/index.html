JavaScript数组操作

常用原生数组操作

  • array.concat(item,…) 将数组浅复制，拼接item（按元素），返回新数组

  • array.join(sep) 将数组中每个元素转换为字符串，用sep把他们连接起来，返回新字符串

  • array.pop() 移除原数组最后一个元素并返回该元素，数组为空则返回undefined

  • array.push(itme,…) 在原数组尾部添加每个item，返回数组的新长度

  • array.reverse() 把原数组里的元素顺序反转

  • array.shift() 移除原数组第一个元素，返回该元素

  • array.unshift(item,…) 将item按参数顺序插入原数组头部，返回数组新长度

  • array.slice(start,end) 从start开始（包含）一直浅复制到end（不包含），返回新数组。参数中有负数则从数组最后开始倒数（最后一个元素位置-1），范围越界则返回空数组。

  • array.splice(dels,count,item,…) 从dels开始（包含）删除count个(未指定则默认剩余所有元素个数，不合法输入则默认转换为0)元素，再把所有的item按顺序插入到dels位置，返回由被删除元素组成的数组。splice最主要用于删除元素。

  • array.sort(fn) 默认不带参数的函数会把待排序的元素视为字符串，所以通常不能正确排序。自定义的比较函数fn接受两个参数，需要符合以下原则：如果两个参数相等则返回0；如果第一个参数应该排在前面，则返回一个负数；如果第一个参数应该排在后面，则返回一个正数。

JavaScript字符串处理方法

常用原生字符串处理函数

  • string.charAt(pos) 返回pos位置的字符组成的字符串，越界则返回空串

  • string.charCodeAt(pos) 返回pos位置字符串的ASCII码，越界则返回NaN

  • string.concat(str,…) 连接字符串，返回新串。最好用+代替，更方便

  • string.indexOf(str,pos) 返回从pos位置（包含）起正向查找匹配str的第一个字符位置，没有则返回-1

  • string.lastIndexOf(str,pos) 与indexOf()类似，逆向查找

  • string.localeCompare(str) 比较字符串大小，大于str返回正数，小于返回负数，相等返回0，但是字符串比较的规则没有详细说明

  • string.match(reg) 与正则表达式匹配，较复杂，将在正则里细说

  • string.replace(str,repl) 支持正则匹配替换
  • string.search(reg) 正则搜索

  • string.slice(start,end) 复制字符串从start开始（包含）到end结束（不包含）的部分，返回新串。起止位置为负数时则表示从串尾开始数（最后一个字符位置是-1）

  • string.split(sep,lim) 按sep分割字符串，用lim限制最多返回的数量。如果seq为空，则返回单字符的数组。支持正则

  • string.substring(start,end) 和slice一样，只是参数不能为负，用slice替代

  • string.toLowerCase() 返回小写的新串
  • string.toUpperCase() 返回小写的新串
  • string.fromCharCode(num,…) 根据一串字符码返回字符串

JavaScript原型链初探

虽然这经常被当作是 JavaScript 的缺点被提及，其实基于原型的继承模型比传统的类继承还要强大。 实现传统的类继承模型是很简单，但是实现 JavaScript 中的原型继承则要困难的多。

使用JavaScript的点和中括号操作符访问对象属性的比较

JavaScript中对象的属性可以用点操作符（.）和中括号操作符（[]）访问，在正常访问和给属性赋值时，两者是等价的。

  • 正常操作时效果一样（访问和赋值）

他们的区别在于：用点操作符的时候，后面必须是一个指定的有效的属性名！所以下面两种场景中，点操作符不会正常运行，而中括号依然有效。

  • 属性名不是合法标识符，点操作符报错，而中括号有效

当属性名是以数字开头或者包含空格或者是包含关键字的无效标识符时，使用点操作符会报错，而中括号依然能访问到，因为中括号强制使用了引号括住属性名，所以可以正常操作如赋值等。

  • 动态设置属性，点操作符无法解析属性名变量，访问无效，而中括号有效

属性名是个变量时，点操作符会把变量当成一个固定的字符串，显然当对象没有名字为该字符串的属性时会返回undefined，而中括号则会正确解析变量，替换为变量值，从而得到正确属性。
