CSS的display常用属性值

发表于 2016-04-16
  • none：不显示，文档流中不存在。（相对来说，visibility:hidden会在文档流中保留原来位置）
  • block：块级元素
    block元素会独占一行，多个block元素会各自新起一行。默认情况下，block元素宽度自动填满其父元素宽度。
    block元素可以设置width,height属性。块级元素即使设置了宽度,仍然是独占一行。
    block元素可以设置margin和padding属性。
  • inline：行内元素
    inline元素不会独占一行，多个相邻的行内元素会排列在同一行里，直到一行排列不下，才会新换一行，其宽度随元素的内容而变化。
    inline元素设置width,height属性无效。
    inline元素的margin和padding属性，水平方向的padding-left, padding-right, margin-left, margin-right都产生边距效果；但竖直方向的padding-top, padding-bottom, margin-top, margin-bottom不会产生边距效果。
  • inline-block：行内块元素
    对象表现为inline，但是内容表现为block。之后的内联对象会被排列在同一行内。
    比如我们可以给a标签设置inline-block值，使其既具有block的宽度高度特性又具有inline的同行特性。
    有点像浮动，但是不从文档流中删除。

CSS伪类和伪元素

发表于 2016-04-16

伪类

锚伪类（CSS1）

LVHA顺序：:link, :visited, :hover, :active。对大小写不敏感。

:first-child 伪类（CSS2）

举例：p:first-child选择DOM中所有的作为第一个子元素的p标签元素。它会先遍历DOM中所有子结构中的第一个元素，判断是不是p元素，然后改变样式。
所以:first-child伪类是指元素作为第一个孩子，而不是指元素的第一个孩子。

:nth-child(n) 选择器（CSS3）

作为first-child的升级版，选择作为某个元素的第n个子元素，n可以是数字或者表达式，如3n+1遍历从0开始的整数n。
举例：p:nth-child(2)，选择DOM中所有元素的第2个子元素，判断如果是p标签则起作用。
同理，:nth-last-child(n)选择某个元素的倒数第n个子元素。
CSS3中还有:last-child和:only-child。
比较有趣的是p:only-child，选择作为独生子的p标签~

:nth-of-type(n) 选择器（CSS3）

和:nth-child()和类似，但是区别在于选择DOM中某个元素的指定类型的子元素中的第n个元素。
举例，p:nth-of-type(2)，选择DOM中所有子结构里的p元素集合里的第2个。
同理，:nth-last-of-type(n)是倒数的。
CSS3中还有:first-of-type、:last-of-type和:only-of-type。

伪元素

:first-line 伪元素（CSS1）

p:first-line给p中的文本首行设置样式。

:first-letter 伪元素（CSS1）

p:first-letter给p中的文本首个字母设置样式。

:before 伪元素

p:before伪元素在p元素的内容前面插入新内容。

:after 伪元素

p:after{
clear:both;
content:”.”;
display:block;
height:0;width:0;
visibility:hidden;
}
元素后插入伪元素，清除浮动。

DOM常用操作

发表于 2016-04-04

操作document元素

  • 创建新节点
    createElement(tagname) //创建一个名字为tagname的元素
    createTextNode(text) //创建一个文本节点，内容为text字符串
    createDocumentFragment() //创建一个DOM片段

  • 添加、移除、替换、插入
    appendChild()
    removeChild()
    replaceChild(newnode,oldnode)
    insertBefore(newItem,existingItem) //在已有的子节点前插入一个新的子节点

  • 查找
    getElementsByTagName() //通过标签名称
    getElementsByName() //通过元素的Name属性的值(IE容错能力较强，会得到一个数组，其中包括id等于name值的)
    getElementById() //通过元素Id，唯一性

  • 父节点
    parentNode

  • 子节点
    firstChild
    lastChild
    children // 节点的直接子节点数组
    childNodes // 节点的所有子节点数组

  • 兄弟节点
    previousSibling
    nextSibling

CSS水平居中和垂直居中

发表于 2016-04-04

垂直居中

  • 最土的表格法

使用一行一列的表格，把div内容放到td里，利用td的vertical-align属性垂直居中。
或者是div表现为单元格，完成水平和垂直居中。

  • 需要较多hack的inline-block法
    基本思路是在父容器里设置行高line-height等于整体高度height
  • 绝对定位
    最早的实现方法需要知道元素宽高。

使用CSS3的transform属性可以操作未知宽高的元素移动相对自身宽高的距离。

  • FlexBox，最好的解决方案
    在需要垂直居中的父元素上设置display:flex并在垂直居中的元素上设置margin:auto，就完成了水平垂直居中。

水平居中

  • 行内元素，在其父元素上设置text-align:center;
  • 块级元素，直接使用margin: 0 auto;
