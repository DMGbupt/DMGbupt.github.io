JavaScript的Object对象

发表于 2015-08-20   |  

JS中所有对象都继承自Object对象，所以不论是普通对象还是函数对象都会继承一些公共的属性和方法，这里记录下比较有用的几个。

可用于所有对象实例的属性和方法

  • constructor属性，指向当前对象的构造函数对象。

a和b都是Object()构造器创建的，所以它们的constructor都指向同一个构造函数对象。F是Function()构造的一个函数对象，它的constructor指向Function()函数对象。

  • hasOwnProperty(Name)方法，判断Name属性是否只存在当前对象中，而不在对象的原型链中。

arr是数组，拥有自己的属性length，但是hasOwnProperty方法却是从Object那继承的，所以是false。

  • hasOwnProperty和for…in结合，可以获取对象自己的所有属性。

  • propertyIsEnumerable(Name)方法，判断Name属性是否可以被for…in枚举。

说明对象的constructor是不可被枚举的。

  • isPrototypeOf(Obj)方法，判断对象是否在obj的原型链上。

foo由F()构造，所以F.prototype在foo的原型链上。

  • toString()方法，返回对象的字符串表示。

  • toLocaleString()方法，同样返回对象的字符串表示，跟所在地区有关。

  • valueOf()方法，返回对象的原始值，可能是字符串、数值或bool值等。
